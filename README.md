# Android Calculator App

A simple and functional calculator app built using **Java** in **Android Studio**. This project demonstrates basic Android development, UI design, and handling user input for arithmetic operations.

## Features
- Basic arithmetic operations: addition, subtraction, multiplication, division
- Clear and reset functions
- Responsive user interface
- Error handling for invalid inputs (e.g., division by zero)
- Supports portrait and landscape orientations

## Technologies
- **Programming Language:** Java
- **Platform:** Android Studio
- **UI Design:** XML layouts
- **Target SDK:** Android API 21+
